# Clinic Queue Manager

A web-based clinic queue management system built using Python Flask.

## Features

- Patient registration
- Queue management
- Call next patient
- Real-time queue display
- Patient statistics

## Technologies Used

- Python
- Flask
- HTML
- CSS
- GitHub version control

## How to Run

1. Install Flask

pip install flask

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
